# System Design Lab

Starter monorepo scaffold for the system design learning and load analysis platform.

## Structure

- `backend/` Python API + orchestration services (FastAPI-oriented)
- `frontend/` React UI (Vite-based starter)
- `shared/` cross-service schemas, examples, and API contracts
- `docs/` requirements, roadmap, and architecture docs
- `scripts/` local helper scripts
